Rhythm Monster

A fast-paced, high-performance rhythm game built with **Godot 4.x**. 
Battle the music by hitting the notes on time using your keyboard!

How to Play
- Use the D, F, J, K keys to hit the notes as they cross the target line.
- Keep your combo high to maximize your score!
- Press *R* to restart if you miss too many notes.

Features
- Custom Draw Engine:** Built using Godot's `_draw` function for ultra-smooth performance and low latency.
- Dynamic Visuals:** Lane flashes and judgment text animations that react to your performance.
- Web Optimized:** Fully compatible with HTML5/Web export for playing in the browser.
- WakaTime Integrated:** Development hours fully tracked and verified.

Technical Details
- Engine: Godot 4.6
- Language: GDScript
- Logic: Beat-based synchronization using the `AudioStreamPlayer` playback position.
